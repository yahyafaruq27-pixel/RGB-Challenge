📚 Mengenal Sistem Warna RGB

RGB adalah model warna aditif di mana cahaya warna Merah (Red), Hijau (Green), dan Biru (Blue) digabungkan dengan berbagai cara untuk mereproduksi susunan warna yang kaya di layar digital.

Setiap komponen warna (channel) memiliki rentang nilai dari 0 (tidak ada cahaya) hingga 255 (kekuatan cahaya penuh). Ini berarti komputer dapat menghasilkan 16,7 juta kombinasi warna unik!

Rumus Campuran Warna Dasar:

  • Merah + Hijau = Kuning (255, 255, 0)
  • Merah + Biru = Magenta (255, 0, 255)
  • Hijau + Biru = Cyan (0, 255, 255)
  • Merah + Hijau + Biru = Putih (255, 255, 255)
  • Semua nilai nol = Hitam (0, 0, 0)
